# 챙기미 제품 기획서

## 한 줄 정의

챙기미는 어르신이 카카오톡 안에서 병원 예약과 복약 일정을 직접 등록·확인하고,
원할 때만 가족에게 공유문을 전달할 수 있게 돕는 본인주도 생활 리마인더 에이전트입니다.

## 문제

떨어져 사는 부모님은 병원 예약과 복약 일정을 깜빡하기 쉽습니다. 병원 예약은
저빈도·불규칙해서 기억 부담이 크고, 놓치면 재예약·진료 공백 같은 실제 손해가 발생합니다.
한편 기존 솔루션은 자녀가 부모를 모니터링하는 구조가 많아, 어르신이 감시당한다는
부담을 느끼게 합니다.

## 해결

어르신 본인이 음성/텍스트로 병원 예약을 등록하고, 약은 사진(약봉투/처방전)으로 등록해
일정을 관리합니다. 가족 공유는 어르신이 원할 때만 켜는 옵트인 방식이며, 공유 범위
변경/해제도 본인이 직접 통제합니다.

## 1차 타깃

병원 예약·복약 일정을 스스로 챙기기 어려운 어르신 본인

## 핵심 차별점

| 일반 캘린더/리마인더 앱 | 챙기미 |
|---|---|
| 새 앱 설치·학습 필요 | 이미 쓰는 카카오톡 안에서 바로 동작 |
| 자녀가 일정을 대신 관리(감시형) | 어르신 본인이 직접 등록·확인(본인주도) |
| 공유가 기본 켜짐 또는 없음 | 옵트인 공유 + 범위 변경/해제 가능 |
| 복약 주의사항을 AI가 새로 설명 | 처방전/약봉투 원문만 그대로 노출 (의료 판단 없음) |

## 왜 카카오인가

- 어르신은 이미 카카오톡을 일상적으로 사용함 — 새로운 학습 곡선이 없음
- 톡캘린더/카카오톡 메시지 등 기존 카카오 인프라와 자연스럽게 결합 가능
- PlayMCP/Kakao Tools를 통해 카카오톡 안에서 바로 노출되는 구조

## 리스크 배제

의료 판단(효능 설명, 복용법 변경, 금식·검사 전 안내 생성), 진단명/검사결과 저장,
가족 직접 발송 선반영, 자체 스케줄러 선반영, 이미지/OCR 선반영, 미성년 자녀 관리,
병원 API/EMR 연동은 다루지 않습니다. (자세한 범위는 [mvp-scope.md](mvp-scope.md))

## 레포 분리 사유

dukpin(K-POP 큐레이션)과 챙기미(어르신 생활 리마인더)는 타깃·데이터 모델·공모전
메시지가 전혀 다르므로 별도 레포로 분리합니다. 코드/지침 공유 없음.
